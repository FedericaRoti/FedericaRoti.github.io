02 — API Integration
NASA Data Explorer

App frontend data-driven che integra le API di NASA (APOD e Mars Weather) con rendering dinamico, modale e visualizzazione dati via Google Charts. Zero framework, JavaScript puro — per capire il DOM prima di delegarlo a React.

"Come si consumano API esterne? Come si rendono i dati in modo leggibile senza un framework?" — Risposta pratica.

JavaScript Fetch API NASA API Google Charts DOM Manipulation
GitHub
03 — Automation + API In corso
InfoBite

Digest quotidiano di notizie tech e AI, costruito con un workflow n8n che aggrega 8 feed RSS, filtra le ultime 24 ore e genera ogni mattina una pagina Notion strutturata. Un frontend React + Vite consuma l'API Notion tramite serverless functions Vercel e li rende liggibili via browser.

"Come si automatizza un flusso di dati reale e lo si espone come prodotto?" — Dati reali, deploy reale, nessun tutorial.

n8n Notion API React Vite Vercel RSS Aggregation Serverless Automation
