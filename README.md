# PORTFOLIO - FEDERICA ROTIROTI

Il mio portfolio personale, costruito da zero con HTML e CSS.
Puoi visitarlo qui -> [FedericaRoti.github.io](https://FedericaRoti.github.io)
---

## Cosa c'è dentro

Single page con presentazione, progetti e form di contatto, più una pagina dedicata alla privacy policy.  
Niente framework, niente dipendenze — solo markup semantico, CSS custom e JS scritto a mano.

Il form di contatto è gestito tramite [Formspree](https://formspree.io).

---

## Tech

- HTML5 semantico
- CSS3 — custom properties, grid, flexbox, animazioni
- JavaScript vanilla — IntersectionObserver per fade-in, nav scroll, hamburger menu mobile
- Formspree per la gestione del form
